# Gravity.io

중력 기반 .io 게임. 먹이를 흡수해 질량을 키우고, 블랙홀을 피하며, 다른 플레이어를 집어삼켜라.

---

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:5173` 접속.

### 모바일 테스트

```bash
npm run dev
# → Network: http://192.168.x.x:5173
```

같은 와이파이에 연결된 폰 브라우저에서 Network 주소로 접속.

---

## 조작법

| 입력 | 동작 |
|---|---|
| 마우스 이동 | 해당 방향으로 추력 |
| Space / Shift | 부스트 (질량 소모) |
| 모바일: 터치 드래그 | 추력 방향 |
| 모바일: BOOST 버튼 | 부스트 |

## 파일 구조

```
src/
├── main.jsx          # React 앱 진입점
├── GravityIO.jsx     # 최상위 UI 컴포넌트 (슬라이더, 점수, 리더보드, 사망화면)
├── useGameLoop.js    # 게임 루프 훅 (RAF, 캔버스 이벤트, init)
├── physics.js        # 물리 업데이트 전체 (중력, AI, 흡수, 감쇄)
├── renderer.js       # 캔버스 렌더링 전체 (블랙홀, 먹이, 웰, HUD, 미니맵)
├── spawners.js       # 오브젝트 생성 함수 (spawnFood, spawnBH, spawnAI)
├── constants.js      # 게임 상수 전체 및 팔레트/이름 목록
└── utils.js          # 토로이달 수학 헬퍼 (wrapDx, wrapDy, di, wrapPos)
```

### 각 파일 역할

#### `constants.js`
모든 수치 상수가 여기 있습니다. 게임 밸런스를 조정할 때 이 파일만 수정합니다.

```
MAP_W/H        맵 크기 (5000×5000)
INITIAL_MASS   시작 질량
THRUST         플레이어 추력
WELL_G         웰(플레이어/AI) 간 중력 상수
FOOD_G         먹이가 웰에 끌리는 힘
BH_MASS        블랙홀 유효 질량
BH_LJ_EQ      블랙홀-먹이 LJ 평형 거리 (먹이 궤도 반지름)
BH_KILL_R      블랙홀 즉사 반경
ABSORB_RATIO   흡수 가능한 질량 비율 (0.75 = 상대가 75% 이하일 때)
BOOST_MULT     부스트 추력 배율
DECAY_RATE     질량 자연 감쇄율
```

#### `utils.js`
맵이 토로이달(상하좌우가 이어진 도넛형)이라 거리 계산에 래핑이 필요합니다.

```
wrapDx/wrapDy  두 점 사이의 최단 방향 벡터 (맵 경계 통과 고려)
di(a, b)       토로이달 거리
wrapPos(obj)   위치를 맵 범위 안으로 래핑
mr(mass)       질량 → 반지름 변환 (√mass × 3)
```

#### `spawners.js`
게임 오브젝트 초기 생성 로직. 블랙홀은 서로 800px 이상 떨어진 위치에 배치됩니다.

#### `physics.js`
매 프레임 호출되는 `updatePhysics(S, dt, W, H, cfg)`.
순서대로 처리합니다:

1. 웰-웰 중력 (플레이어 ↔ AI)
2. 블랙홀 → 웰 인력 + 즉사/워프 처리
3. 블랙홀 → 먹이 LJ 포텐셜 (궤도 형성)
4. 먹이-먹이 인력 (공간 그리드 최적화)
5. 플레이어 추력 / 부스트
6. AI 행동 (도주·추격·먹이 탐색)
7. 먹이 → 웰 인력
8. 흡수 판정 (먹이, 블랙홀, 웰-웰)
9. 먹이 리스폰 / 질량 감쇄 / FX 파티클

#### `renderer.js`
매 프레임 호출되는 `render(ctx, S, W, H, cfg)`.

- **카메라**: 플레이어 토로이달 스무스 팔로우, 질량에 따라 줌 아웃
- **블랙홀**: 이벤트 호라이즌 그라데이션, LJ 링, 회전 광점
- **중력선**: 가까운 웰 사이에 베지어 곡선으로 시각화
- **HUD**: 위험 인력 바, 화면 밖 오브젝트 화살표, 속도 표시
- **미니맵**: 우측 하단, 전체 맵 축소 뷰

#### `useGameLoop.js`
React 커스텀 훅. `started` 상태가 true가 되면 캔버스 이벤트를 등록하고 RAF 루프를 시작합니다. 컴포넌트 언마운트 시 정리합니다.

#### `GravityIO.jsx`
순수 UI 레이어. 게임 로직 없음.

- 시작 화면: 파라미터 슬라이더 6개 (로그 스케일, 기본값 pos=50)
- 인게임: 점수, 리더보드, 부스트 버튼(모바일), 처음으로 버튼
- 사망 화면: 최종 질량, 다시하기 / 처음으로

## 파라미터 슬라이더

시작 화면에서 조정 가능. 슬라이더는 로그 스케일 (중간=기본값, 좌=÷1000, 우=×1000).

| 슬라이더 | 변수 | 기본값 | 효과 |
|---|---|---|---|
| 마찰 | drag | 1.2 | 높을수록 빨리 감속 |
| 추력 | thrust | 320 | 플레이어·AI 이동력 |
| 먹이 인력 | foodG | 30 | 먹이 클러스터 밀도 |
| 웰 중력 | wellG | 3000 | 웰 간 인력 세기 |
| 블랙홀 | bhMass | 300 | 블랙홀 인력 세기 |
| 먹이 수 | foodCount | 1400 | 맵 내 먹이 총량 |
